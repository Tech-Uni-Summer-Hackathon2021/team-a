## プロダクト名
***smart adjust***
## プロダクト説明
*このwebサービスによって生徒同士の仲が深まることを目的としています。*
*そして、仲を深めるためのイベントの企画を簡略化します。*
*具体的には、このwebサービスがイベント幹事の役割を担いメンバーの決定から日程やイベント内容の決定まであらゆる手間のかかる作業を行ってくれます。*
## 使用技術
フロントエンド： Vue.js（Nuxt.js）　/ Vuetify
バックエンド: Rails(apiサーバー)
DB: SQLite
インフラ： Docker
